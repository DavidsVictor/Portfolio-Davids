# Meu Portfólio Profissional 🚀

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para demonstrar minhas habilidades em desenvolvimento web e servir como uma vitrine dos meus projetos.

## 🛠️ Tecnologias Utilizadas
* **HTML5** e **CSS3** (Estrutura e estilização personalizada)
* **JavaScript** (Interatividade e animações)
* **VS Code** (Editor de código)

Este portfólio apresenta minha trajetória como desenvolvedor Full Stack em formação.
